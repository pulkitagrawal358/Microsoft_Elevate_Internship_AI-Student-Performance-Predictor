AI-Powered Student Performance Prediction System

Project Overview
Developed during the Microsoft Elevate - AICTE Internship (Jan 2026 Batch), this project addresses the "visibility gap" in academic outcomes. By leveraging Machine Learning, the system analyzes student behavioral data—such as study hours, attendance, and parental support—to predict academic performance levels. This serves as an early-warning tool for educators to identify at-risk students and provide data-driven interventions.

Features
* **Predictive Modeling: Uses a Random Forest Classifier to categorize student performance into Grade Classes (0-4).
* Data-Driven Insights: Identifies the most impactful factors affecting a student's GPA.
* Scalable Design: Conceptual architecture designed for deployment on Microsoft Azure.

Tech Stack
* Language: Python
* Libraries: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
* Environment: Google Colab / Jupyter Notebook
* Dataset: Student Performance Dataset (Kaggle)
